# Re-Balans React Site

Минимальный проект Vite + React для деплоя на Vercel.

## Локально
npm install
npm run dev

## Деплой
1. Загрузите папку в GitHub
2. В Vercel нажмите Add New Project
3. Выберите репозиторий
4. Нажмите Deploy

Tailwind подключён через CDN в index.html, чтобы проект собрался без отдельной настройки Tailwind.
